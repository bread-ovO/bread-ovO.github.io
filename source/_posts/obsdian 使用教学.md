## 新建仓库

Obsidian 支持「仓库」概念，我们可以将自己的「个人笔记」和「工作笔记」创建不同的仓库，这样他们之间的内容互不影响，当然一切都是自由的，你可以任意创建属于自己的「仓库」。

## 插件

obsdian 最吸引我的一点就是有各种各样的插件，而这些插件可以给自己带来非常舒适的体验😋~~（不要被插件插坏了喂！）~~

### 1.[盘古 PanGu](https://github.com/Natumsol/obsidian-pangu)
自动为中英文之间插入空格，排版强迫症的福音！
（虽然会手动排版， 但是有的时候忘记了是真的很烦）

### 2.[advanced tables](https://github.com/tgrosinger/advanced-tables-obsidian)
高级版 Obsidian 表格，它添加了改进的导航、格式化和操作功能到 Obsidian 中的 Markdown 表格
![[Pasted image 20241210090440.png]]
![[Pasted image 20241210090452.png]]

| Syntax    | Description |
| --------- |:----------- |
| Header    | Title       |
| Paragraph | Text        |
|           |             |
### 3.[Auto Link Title](https://github.com/zolrath/obsidian-auto-link-title)
这个可以插件自动抓取网页以提取粘贴时的链接标题，创建带有正确标题的 Markdown 链接。
（就是有的时候标题太长会出来一大坨🥹）

### 4.[Editing Toolbar](https://github.com/PKM-er/obsidian-editing-toolbar)
Obsidian 编辑工具栏插件,就是顶部的编辑栏位，有的时候还挺好用的😋
![[Pasted image 20241210091731.png]]

### 5.[Git](https://github.com/Vinzent03/obsidian-git)
伟大，无需多言，在 obsdian 官方支持的云同步要马内的情况下，github简直是慈善（
使用详情请见文档。
该说不说啊用这个来多端同步也挺好用的，不知道还能不能找到更好的方式

### 6.[Quiet Outline](https://github.com/guopenghui/obsidian-quiet-outline)
使标题更容易阅读的插件
[使用文档在这](https://github.com/guopenghui/obsidian-quiet-outline/blob/master/README-CN.md)
支持搜索！好用！

### [7.Rollover Daily Todos](https://github.com/lumoe/obsidian-rollover-daily-todos)
这个 Obsidian 插件将自动将前一天的每日笔记中未完成的待办事项（可能是昨天或一周前）滚动到今天。当通过内部 `Daily notes` 插件创建新的每日笔记时，此功能会自动触发。它还可以从命令面板中作为命令运行。非常好用，生产力）

### [8. Task Progress Bar](https://github.com/Quorafind/Obsidian-Task-Progress-Bar)
用于在任务项目符号或标题附近显示任务进度条,非常好用，配合上一个插件很好用。
